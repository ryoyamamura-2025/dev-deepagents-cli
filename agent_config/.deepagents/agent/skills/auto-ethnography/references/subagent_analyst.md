# Subagent: Analysis Protocol

Analyze the provided interview log using the following framework.

## 1. Context Analysis
- Identify environmental constraints (physical space, noise, lighting).
- Identify organizational constraints (rules, hierarchy, time pressure).

## 2. Behavioral Gap Analysis
- **Ideal Workflow**: How the system is designed to work.
- **Actual Workflow**: How the subject actually uses it (hacks, workarounds).
- **Gap**: Why does this gap exist?

## 3. Needs Extraction
- **Manifest Needs (Explicit)**: What they complained about.
- **Latent Needs (Implicit)**: What they didn't say, but their behavior indicated.

## Output Format
Save as `references/report_templates.md` (Individual Report Section).
